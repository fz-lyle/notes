## Canvas

```html
<body>
    <canvas id="canvas" width="500" height="500"></canvas>
</body>
```

```js
// 获取指定的canvas元素
let canvas = document.getElementById('canvas')
// 调用canvas的 getContext 方法访问获取2d渲染上下文
let ctx = canvas.getContext('2d')
// 通过2d上下文执行绘画操作来绘制图形
ctx.绘画操作();
```
